An official highly optimized, fully responsive modern Bio Link and Interactive Digital Portfolio built with **React**, **Vite (TypeScript)**, **Tailwind CSS v4**, and **Motion** animations.

Developed with a sophisticated, cyberpunk-infused **Dark Lightning Aesthetic** specifically designed for showcasing creative, digital-focused academic paths, technical experiences, and credentials.

---

## ✨ Features & Highlights

- 🌀 **Interactive Canvas Background**: dynamic physical lightning vector simulation drawing particle coordinates that interact with browser viewport movements.
- 🎓 **Formal Education Timeline**: Fully detailed timeline layout showing academic tracks from *SDIT Al Uswah Surabaya* up to *SMA Negeri 17 Surabaya* with precise achievement highlights.
- 🏆 **Dynamic Achievements Grid**: Curated selection of prestigious achievements such as Gold Medalist at *World Youth Invention and Innovation Award 2023 (WYIIA)*, *Puspresnas Bina Talenta BTI 2025*, and *ITS GIGA Lab GTC*, complete with instant cloud certificate verification access.
- 🎨 **Premium Aesthetic & Pairing**: Built using high-contrast slate canvases combined with glowing ultraviolet and lime-accent energy indicators. Utilizes precise responsive view switches (desktop menu bar to intuitive modern bottom/drawer drawers).
- 🚀 **High Performance Core**: Lightning-fast load times with standard ESM bundling, minimal bundle weight, and strict TypeScript compilation.

## 🚀 Getting Started

Follow these instructions to set up, preview, and run the project locally.

### Prerequisites

You must have **Node.js (v18+)** and **npm** installed on your system.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/dejavi08-biolink.git
   cd dejavi08-biolink
   ```

2. **Install all dependencies:**
   ```bash
   npm install
   ```

3. **Run the local development server:**
   ```bash
   npm run dev
   ```
   *Your terminal will expose the local address (typically `http://localhost:3000`). Make sure port 3000 is open.*

### Building for Production

To compile files and output static production assets, run:
```bash
npm run build
```
The optimized bundle will be compiled and organized into the `/dist` directory, completely ready to host on Vercel, Netlify, GitHub Pages, or Cloud Run containers.
